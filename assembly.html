What is Assembly?

Assembly is any human readable programming language that is strongly comparable to machine code instructions.

It is converted to machine code using a program called an "assembler", which tailors the output depending on the instruction set of the intended processor.

Many languages use assembly as an intermediate step, in which the program can be optimised, before assembling to machine code - this is usually easier than creating a compiler for every architecture.

Little Man Computer

The LMC is a proof of concept simulator that uses a custom interpreted assembly language. It has instructions:

  1. ADD - add
  2. SUB - subtract
  3. STA - store
  4. LDA - load
  5. BRA - branch
  6. BRZ - branch if zero
  7. BRP - branch if positive
  8. INP - input value
  9. OUT - output value
  10. HLT - end program
  11. DAT - data location
